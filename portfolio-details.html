The Mandalorian: Ashes of the Empire

Project information

  • Category: Game
  • Project date: June 2021
  • Project URL: Website of the project

The Mandalorian: Ashes of the Empire is an action fast-paced 3D Roguelite game. This project was done in three months by 32 video game development students using our own c++ game engine for the subject "Project 3" CITM, UPC.

My main position within the design department was as narrative and level designer. During the project I've designed levels, designed and prototiped enemies and one of the bosses, designed and developed the tutorial, designed the skill trees and developed the narrative and the dialog system.
